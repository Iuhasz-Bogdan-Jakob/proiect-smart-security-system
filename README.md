# Proiect Smart Security System
Sistem de Securitate IoT pentru Casă

Acest proiect reprezintă un sistem de securitate inteligent bazat pe platforma Arduino, conceput pentru a monitoriza mișcarea, prezența gazelor periculoase și starea punctelor de acces (uși/ferestre). Sistemul integrează tehnologii IoT pentru a trimite notificări în timp real direct pe smartphone-ul utilizatorului prin intermediul aplicației PushBullet folosind un API custom pentru fiecare user înregistrat pe această aplicație. S-a optat pentru utilizarea unei aplicații de tip push notifications gratuită pentru a reduce costurile creării unui astfel de sistem.
